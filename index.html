How to Apply CSS to HTML

If you're wondering on how to include an external CSS file into your HTML code, it's time to listen up. Make sure you know the path to your CSS file relative to your HTML file. Once you know that, all you need to do is link the file path in your HTML code inside your <head></head> element.

For example: 

	<!DOCTYPE html>
	<html>
		<head>
			<link rel="stylesheet" href="./resources/style.css">
		</head>
	</html>

In this example, the path to the file would be "./resources/style.css". Once this your CSS file is linked to your HTML, all styling should be applied!

How to Use JavaScript in a Webpage

Now that we learned how to include an external CSS file into your HTML code, let's try to do the same with a JavaScript. Much of the same principles apply; however, instead of using a <link> element, we'll use the <script> element inside of the <body> element.

For example: 

	<!DOCTYPE html>
	<html>
		<body>
			<script src="./resources/script_src.js">
			</script>
		</body>
	</html>

As you can see, we are using the src attribute instead of the href attribute we used for linking the CSS file. Assuming you're JavaScript file has something in it, it should now show up on the page! But what if we wanted to add some JavaScript inside our HTML without using an external file? Well, we can do that too! Similarly to the external JavaScript file, we'll be using the <script> element inside of the <body> element. Instead of having your JavaScript in its own file, we'll put it right inside the HTML code.

For example: 

	<!DOCTYPE html>
	<html>
		<body>
			<h1 id="test">Place Holder Text</h1>
			<script>
				document.getElementById("test").innerHTML = "My Cool Webpage";
			</script>
		</body>
	</html>

As you can see, this should replace the text in the <h1> with something a little more suited for your webpage.
